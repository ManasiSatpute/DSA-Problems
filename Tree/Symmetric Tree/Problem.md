# 101. Symmetric Tree

## Problem Summary

Given the root of a binary tree, determine whether the tree is symmetric around its center.

A tree is symmetric if the left subtree is a mirror image of the right subtree.

## Example

### Example 1

Input:

        1
      /   \
     2     2
    / \   / \
   3   4 4   3

Output:

true

### Example 2

Input:

        1
      /   \
     2     2
      \     \
       3     3

Output:

false

## Constraints

- Number of nodes is in the range **[1, 1000]**
- Node values are between **-100 and 100**
