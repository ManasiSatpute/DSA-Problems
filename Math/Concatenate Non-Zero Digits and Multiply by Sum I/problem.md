# Concatenate Non-Zero Digits and Multiply by Sum I

Given an integer, process its digits while ignoring every occurrence of `0`.

The remaining non-zero digits should retain their original order to form a new integer. After constructing the new integer, perform the required computation as specified in the problem statement.

## Example

Input

1203405

Processing

* Digits extracted: 5, 0, 4, 3, 0, 2, 1
* Ignore zeros
* Remaining digits: 5, 4, 3, 2, 1
* Restore original order
* New number: 12345

The final answer is obtained by applying the required operation on this newly formed number.
