# Problem: GCD of Odd and Even Sums

## Problem Statement

Given an integer `n`:

- Find the sum of the first `n` odd numbers.
- Find the sum of the first `n` even numbers.
- Return the Greatest Common Divisor (GCD) of these two sums.

## Example

### Input

```
n = 4
```

### Calculation

First 4 odd numbers:

```
1 + 3 + 5 + 7 = 16
```

First 4 even numbers:

```
2 + 4 + 6 + 8 = 20
```

GCD:

```
gcd(16, 20) = 4
```

### Output

```
4
```
