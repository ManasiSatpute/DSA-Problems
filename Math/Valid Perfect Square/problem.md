# Valid Perfect Square

##  Problem Statement

Given a positive integer `num`, write a function to determine if it is a perfect square or not.

A **perfect square** is an integer that is the square of an integer. In other words:

n = k × k

---

##  Examples

### Example 1:
Input:
num = 16

Output:
true

Explanation:
4 × 4 = 16

---

### Example 2:
Input:
num = 14

Output:
false

Explanation:
No integer squared equals 14

---

##  Constraints

- 1 <= num <= 2³¹ - 1
