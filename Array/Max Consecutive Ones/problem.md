# Max Consecutive Ones

## Problem Statement
Given a binary array `nums`, return the **maximum number of consecutive 1's** in the array.

A binary array contains only `0` and `1`.

---

## Examples

### Example 1
Input:
nums = [1,1,0,1,1,1]

Output:
3

Explanation:  
The first two digits or the last three digits are consecutive `1`s.  
The maximum number of consecutive `1`s is **3**.

---

### Example 2
Input:
nums = [1,0,1,1,0,1]

Output:
2

Explanation:  
The longest streak of `1`s appears in the middle with length **2**.

---

## Constraints

- `1 <= nums.length <= 10^5`
- `nums[i]` is either `0` or `1`.
