# LeetCode 3120 - Count the Number of Special Characters I

## Problem Statement

You are given a string `word`.

A letter is called **special** if it appears **both in lowercase and uppercase** in the string.

Return the number of such special characters.

---

## Example

### Example 1:
Input: word = "aaAbcBC"  
Output: 3  
Explanation: 
- 'a' appears in both lowercase and uppercase
- 'b' appears in both lowercase and uppercase
- 'c' appears in both lowercase and uppercase

---

## Constraints
- 1 ≤ word.length ≤ 1000
- word consists of lowercase and uppercase English letters only
