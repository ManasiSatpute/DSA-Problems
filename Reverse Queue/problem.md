# Reverse a Queue Using Stack

## Problem Statement
Given a queue of integers, your task is to **reverse the order of its elements** using an auxiliary data structure.
You must modify the original queue such that the first element becomes the last and the last element becomes the first.
---
## Input
- A queue `q` containing `n` integers
---
## Output
- The same queue with its elements reversed
---

## Example
### Input
1 2 3 4 5

### Output
5 4 3 2 1
