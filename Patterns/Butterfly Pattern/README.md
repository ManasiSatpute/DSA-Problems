# Butterfly Pattern

A classic pattern-printing problem that demonstrates the use of nested loops and symmetry in programming.

## Problem Overview

Print a butterfly-shaped pattern using stars (`*`) for a given integer `n`.

### Example Output (`n = 4`)

```text
*      *
**    **
***  ***
********
********
***  ***
**    **
*      *
```

## Concepts Used

* Nested Loops
* Pattern Printing
* Space Management
* Symmetry
* Iterative Programming

## Time Complexity

```text
O(n²)
```

## Space Complexity

```text
O(1)
```

## Learning Outcome

Through this problem, I practiced:

* Breaking a pattern into smaller sections.
* Controlling rows and columns using loops.
* Managing spaces and alignment.
* Improving logical thinking for pattern-based questions.
