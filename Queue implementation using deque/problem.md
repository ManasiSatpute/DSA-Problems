# Implement Queue Using Deque (C++)

## Problem Statement

Implement a **Queue** data structure using **C++ `deque`**.  

Your queue should support the following operations:

- **push(x)**: Insert element `x` at the end of the queue  
- **pop()**: Remove the element at the front of the queue  
- **front()**: Return the element at the front of the queue  
- **empty()**: Check if the queue is empty

---

## Input

- A sequence of operations to perform on the queue

---

## Output

- Output elements when dequeuing the queue using `pop()` and `front()` operations

---

### Input
```text
Push 1, 2, 3, 4, 5
